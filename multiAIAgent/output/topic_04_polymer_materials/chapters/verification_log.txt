【例2】粘弾性モデル - Maxwell模型とVoigt模型
================================================================================

実行コマンド:
$ python example2_viscoelastic_models.py

実行結果: ✅ 成功

主な出力:
- Maxwell模型（応力緩和）:
  * 模型1: E = 1.0 GPa, η = 1.0×10^10 Pa·s, τ = 10.00 s
  * 模型2: E = 1.0 GPa, η = 1.0×10^11 Pa·s, τ = 100.00 s

- Voigt模型（クリープ）:
  * 模型1: E = 1.0 GPa, η = 1.0×10^10 Pa·s, τ = 10.00 s
  * 模型2: E = 1.0 GPa, η = 1.0×10^11 Pa·s, τ = 100.00 s

生成ファイル:
✅ maxwell_stress_relaxation.png (応力緩和曲線、線形・対数スケール)
✅ voigt_creep.png (クリープコンプライアンス曲線)
✅ maxwell_dynamic.png (動的粘弾性特性、複素弾性率・損失正接)

検証項目:
✅ Maxwell模型の応力緩和関数 σ(t) = ε0·E·exp(-t/τ)
✅ Voigt模型のクリープ関数 ε(t) = (σ0/E)·(1-exp(-t/τ))
✅ 複素弾性率の計算（貯蔵弾性率E'、損失弾性率E"）
✅ 損失正接 tan(δ) = E"/E' = 1/(ωτ)
✅ 緩和時間τ = η/E の関係

物理的妥当性:
✅ Maxwell模型: 長時間で応力→0（液体的挙動）
✅ Voigt模型: 長時間で一定ひずみ（固体的挙動）
✅ ωτ = 1 付近で E' ≈ E" （エネルギー散逸が最大）
✅ 緩和時間が長いほど、ゆっくりと緩和・クリープ

実行結果: ✅ 成功

主な出力:
- PS/PMMAブレンド (50/50):

  * Gordon-Taylor (k=1.0): Tg = 102.50°C
  * Gordon-Taylor (k=1.2): Tg = 102.73°C

- スチレン-ブタジエン共重合体 (25% St):
  * Fox式: Tg = -75.21°C
  * Gordon-Taylor (k=0.5): Tg = -21.20°C

- 予測精度比較:
  * Fox式の平均誤差: 5.12 K
  * Gordon-Taylor式の平均誤差: 13.26 K
  （注: Gordon-Taylor式は適切なkパラメータ使用時に高精度）

生成ファイル:
✅ tg_composition_ps_pmma.png (PS/PMMAのTg-組成曲線)
✅ tg_composition_sbr.png (SBRのTg-組成曲線、実験データ付き)
✅ tg_prediction_accuracy.png (予測精度の比較バーグラフ)

検証項目:
✅ Fox式: 1/Tg = w1/Tg1 + w2/Tg2
✅ Gordon-Taylor式: Tg = (w1·Tg1 + k·w2·Tg2) / (w1 + k·w2)
✅ kパラメータの逆算機能
✅ 組成全範囲でのTg変化の可視化
✅ 複数の高分子系での精度評価

物理的妥当性:
✅ Tgは純成分の値の間に収まる（ほぼ全ての場合）
✅ kパラメータで実験値へのフィッティングが可能
✅ Tg差が大きい系（SBR）で両式の差が顕著
✅ 実験データとの比較が適切
