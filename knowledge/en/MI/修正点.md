# 修正点

Mermaid v10.9.4 ではサブグラフ ID に英数字とアンダースコアのみが使用可能です。ハイフンやスペースを含む ID が残っていると、ブラウザ側で `Syntax error in text` が発生します。以下の 5 箇所で不正な ID を確認しました。

1. `active-learning-introduction/chapter-1.html:628`
   - 現状: `subgraph Exploration-focused`
   - 問題: ID にハイフンが含まれている
   - 修正案: `subgraph Exploration_Focused [Exploration-focused]`

2. `active-learning-introduction/chapter-1.html:634`
   - 現状: `subgraph Exploitation-focused`
   - 問題: ID にハイフンが含まれている
   - 修正案: `subgraph Exploitation_Focused [Exploitation-focused]`

3. `bayesian-optimization-introduction/chapter-1.html:1146`
   - 現状: `subgraph Exploration-focused`
   - 問題: ID にハイフンが含まれている
   - 修正案: `subgraph Exploration_Focused [Exploration-focused]`

4. `bayesian-optimization-introduction/chapter-1.html:1152`
   - 現状: `subgraph Exploitation-focused`
   - 問題: ID にハイフンが含まれている
   - 修正案: `subgraph Exploitation_Focused [Exploitation-focused]`

5. `bayesian-optimization-introduction/chapter-2.html:1221`
   - 現状: `subgraph Optimal Balance`
   - 問題: ID にスペースが含まれている
   - 修正案: `subgraph Optimal_Balance [Optimal Balance]`

各サブグラフの `end` ブロック内で参照するノードやスタイルはそのまま利用できます。識別子を英数字+アンダースコアに変更し、表示名を角括弧で指定することで、Mermaid 10.9.4 でもエラーなく描画されます。
