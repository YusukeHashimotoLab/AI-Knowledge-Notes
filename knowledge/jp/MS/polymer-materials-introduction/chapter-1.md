### 1.4.1 æ•°å¹³å‡åˆ†å­é‡ã¨é‡é‡å¹³å‡åˆ†å­é‡

é«˜åˆ†å­ã¯åˆ†å­é‡åˆ†å¸ƒã‚’æŒã¤ãŸã‚ã€å¹³å‡åˆ†å­é‡ãŒè¤‡æ•°å®šç¾©ã•ã‚Œã¾ã™ã€‚æœ€ã‚‚é‡è¦ãªã®ã¯**æ•°å¹³å‡åˆ†å­é‡ï¼ˆMâ‚™ï¼‰** ã¨**é‡é‡å¹³å‡åˆ†å­é‡ï¼ˆMwï¼‰** ã§ã™ã€‚

### 1.5.1 æœ€ç¢ºåˆ†å¸ƒï¼ˆMost Probable Distributionï¼‰

ç¸®åˆé‡åˆï¼ˆé€æ¬¡é‡åˆï¼‰ã§ã¯ã€åˆ†å­é‡åˆ†å¸ƒã¯**Flory-Schulzåˆ†å¸ƒ** ï¼ˆæœ€ç¢ºåˆ†å¸ƒï¼‰ã«å¾“ã„ã¾ã™ã€‚ã“ã‚Œã¯ã€å®˜èƒ½åŸºã®åå¿œãŒç„¡å·®åˆ¥ã«èµ·ã“ã‚‹ã¨ã„ã†ä»®å®šã«åŸºã¥ãçµ±è¨ˆçš„åˆ†å¸ƒã§ã™ã€‚ 

é‡åˆåº¦ \\(n\\) ã‚’æŒã¤åˆ†å­ã®ãƒ¢ãƒ«åˆ†ç‡ \\(N_n\\) ã¯ã€å®˜èƒ½åŸºè»¢åŒ–ç‡ \\(p\\) ã‚’ç”¨ã„ã¦ä»¥ä¸‹ã®ã‚ˆã†ã«è¡¨ã•ã‚Œã¾ã™ï¼š 

\\[ N_n = n p^{n-1} (1 - p)^2 \\] 

ã“ã®åˆ†å¸ƒã‹ã‚‰ã€ä»¥ä¸‹ã®å¹³å‡å€¤ãŒå°ã‹ã‚Œã¾ã™ï¼š 

  * æ•°å¹³å‡é‡åˆåº¦: \\(\overline{DP_n} = \dfrac{1}{1 - p}\\)

ä¾‹ãˆã°ã€\\(p = 0.99\\) ã®å ´åˆã€PDI = 1.99 â‰ˆ 2 ã¨ãªã‚Šã¾ã™ã€‚ç¸®åˆé‡åˆã§ã¯ç†è«–çš„ã« PDI â‰¥ 2 ã¨ãªã‚Šã€åºƒã„åˆ†å­é‡åˆ†å¸ƒã‚’æŒã¡ã¾ã™ã€‚ 

#### Pythonå®Ÿè£…: Flory-Schulzåˆ†å¸ƒã®ã‚·ãƒŸãƒ¥ãƒ¬ãƒ¼ã‚·ãƒ§ãƒ³

    def flory_schulz_distribution(n, p):
        """
        Flory-Schulzåˆ†å¸ƒï¼ˆæœ€ç¢ºåˆ†å¸ƒï¼‰ã‚’è¨ˆç®—
        
        Parameters:
        -----------
        n : int or array
            é‡åˆåº¦
        p : float
            å®˜èƒ½åŸºè»¢åŒ–ç‡
        
        Returns:
        --------
        N_n : float or array
            ãƒ¢ãƒ«åˆ†ç‡
