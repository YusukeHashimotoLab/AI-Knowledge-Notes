偏微分方程式と境界値問題

Partial Differential Equations and Boundary Value Problems

物理・工学で最も重要な偏微分方程式（波動方程式、熱伝導方程式、ラプラス方程式）の解析的・数値的解法を学びます。変分法、有限要素法、有限差分法など実用的な数値解法も含み、材料プロセスシミュレーションへの応用を通じて実践的なスキルを習得します。

📚 5章構成 💻 40コード例 ⏱️ 100-120分 📊 中級

🎯 学習目標

🌊

古典的PDEの理解

波動方程式、熱伝導方程式、ラプラス方程式の性質と解法を習得する

🎼

変数分離法とフーリエ展開

境界値問題を変数分離法で解き、固有関数展開を理解する

∫

変分法と最適化

汎関数の極値問題とオイラー・ラグランジュ方程式を理解する

🔢

数値解法の実装

有限差分法、有限要素法でPDEを数値的に解く技術を身につける
